Aplicación de mantenimiento de usuarios, esta aplicación
permite realizar las operaciones CRUD básicas a la entidad
UserModel (users), el password es encriptado y almacenado a la 
base de datos (al actualizar la contraseña se vuelve a encriptar 
en la base de datos)
